About The Job
The Application Security Director will be responsible for demonstrating Jet’s dedication to the security and privacy of our customer’s data by leading programs that ensure that Jet continues to develop secure software, all while continuing to create innovate solutions that provide meaningful savings to our customers.
About The Role
· Lead the Application Security team responsible for regular application security assessments
· Implement a process for an end-to-end application security program using automated tools and manual techniques to identify and verify exposure to common security vulnerabilities and provide remediation guidance
· Act as source of and/or conduit to information security subject matter expertise in Application Security
· Perform technical security assessments, secure code audits, and design reviews
· Advocate for security and secure coding practices throughout Jet
· Ensure the ongoing integration of application security with business strategies and requirements 
About You
· 5 - 10 years of experience in information security, preferably with some experience in application development or securing web applications
· Deep understanding of information technology and application security practices, including the areas of platform architecture, application security modeling, and systems integrity
· Subject matter expert in the tools used to build an Application Security program
· Experience leading an application security team at a dynamic, innovative company
· Extraordinarily technical and able to do hands on work when needed
· Knowledgeable about secure development life-cycle, threat modeling, and web application security assessments
